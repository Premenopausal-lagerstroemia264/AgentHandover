## 🧭 What AgentHandover Does

AgentHandover helps your AI agents learn from the work you already do.

It watches your actions, captures patterns, and turns them into guidance that your agents can reuse. That means less repeated setup, fewer plain prompts, and more work done the way you like it.

Use it when you want tools like OpenClaw, Claude Code, or Codex to follow your style without full manual instructions each time.

## ✨ What You Can Use It For

- Teach agents how you name files and folders
- Record repeat steps in your daily workflow
- Reduce the time spent explaining the same task
- Help agents follow your common rules
- Build a growing set of reusable skills
- Keep your process more consistent across tasks

## 💡 How It Works

AgentHandover uses a simple loop:

1. It observes what you do
2. It finds repeat steps
3. It turns those steps into a skill
4. It gives that skill to your agent
5. Your agent uses it next time

This helps your agents move from one-off answers to repeatable work habits.

## 🧭 First-Time Use

### 1. Pick one task

Start with one task that you do often. Good examples are:

- Renaming files
- Moving folders
- Preparing project folders
- Repeating prompt patterns
- Sorting task output

### 2. Work the way you usually work

Do the task in your normal way. AgentHandover learns from real habits, not from made-up examples.

### 3. Let the app capture the flow

Keep the app open while you work. It uses that session to learn what matters.

### 4. Check the learned skill

Review the saved skill or handover record after the task ends. Make sure it reflects your real process.

### 5. Reuse it with your agent

Send the learned skill to your agent tool and use it on the next task.

## 🧠 Best Results

AgentHandover works best when you:

- Keep tasks focused
- Use the same steps each time
- Avoid changing the process mid-task
- Teach one workflow at a time
- Review the learned output before reuse

Clear steps lead to better handover data.

## ❓ Troubleshooting

### The file will not open

Make sure you downloaded the Windows file from the release page and not the source code archive.

### Windows shows a security prompt

Choose the option to run the app if you trust the release file you downloaded.

### The app opens and closes fast

Try downloading the file again. A broken download can stop the app from starting.

### Nothing seems to be learned

Use a task with clear repeat steps. Very short or random actions may not create useful handover data.

### The app does not fit my workflow

Try teaching one smaller task first. A narrow task often works better than a broad one.
